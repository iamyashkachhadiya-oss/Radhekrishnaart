# Radhe Krishna Art

Luxury machine embroidery craftsmanship since 2008.

## About the Project
This website showcases the premium embroidery services of Radhe Krishna Art, based in Surat, Gujarat. It features high-end design inspired by luxury aesthetics, interactive elements, and a dedicated showcase of our production power.

## Key Features
- **Luxury Aesthetic**: Deep black and gold palette.
- **The 14-Machine Mill**: Immersive production showcase.
- **Responsive Design**: Optimized for all devices.
- **Personalized Story**: Highlighting the legacy of Nitinbhai, Nilesbhai, and Pareshbhai.

## Tech Stack
- HTML5
- Vanilla CSS
- Vanilla JavaScript
